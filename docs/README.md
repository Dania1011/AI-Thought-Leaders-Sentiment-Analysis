The **AI Thought Leaders Sentiment Analysis Platform** is a complete data engineering and sentiment analysis pipeline designed to collect, clean, process, and analyze social media posts from leading AI researchers such as Geoffrey Hinton, Ilya Sutskever, and Andrej Karpathy.  

The system uses **Nitter-based scraping**, **data cleaning**, **sentiment scoring using VADER**, and **user-level comparative analysis**, generating actionable insights into public communication trends of top AI personalities.
